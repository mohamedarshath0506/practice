# Heading

HTML headings defined with the `<h1>` to `<h6>` tags.

HTML heading with the `style` attribute using the CSS `font-size` specify the size.

```html

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

```

```css
<h1 style="font-size:60px;">Heading 1</h1>
```
